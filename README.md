# 🚀 DigitalOrbit – Modern Digital Agency Website

DigitalOrbit is a fully responsive and modern digital agency website built using HTML5, CSS3, and Bootstrap 5.  
It is designed with a futuristic gradient theme and smooth UI components.

---

## 🌟 Live Preview

You can open the `index.html` file in your browser to view the project locally.

---

## 📌 Features

✔ Responsive Navigation Bar  
✔ Hero Section with Animation  
✔ About Section  
✔ Services Section  
✔ Portfolio Gallery  
✔ Testimonials Carousel  
✔ Contact Form  
✔ Modern Footer  
✔ Gradient & Glassmorphism Design  
✔ Fully Responsive Layout

## 🚀 How To Run Locally

1. Download or Clone this repository.
2. Open the project folder.
3. Double-click on `index.html`.
4. The website will open in your default browser.

---

## 🔧 How To Customize

- Edit `style.css` to change colors and theme.
- Modify content inside `index.html`.
- Replace images inside the `image/` folder.
